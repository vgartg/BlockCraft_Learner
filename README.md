# Образец игры для изучения Python с использованием pygame

## Разделы для редактирования

#### /mods - Самый базовый раздел, простые редактирования для тестирования возможностей игры
#### /game - Внутренняя логика игры, более сложный уровень для редактирования
#### /assets - Ассеты для игры. На текущий момент только текстуры

## Демонстрация

## Управление

#### Движение вперед/назад/влево/вправо - стрелочками ЛИБО клавишами W S A D, соответственно
#### Прыжок - Space
#### Сломать блок - ЛКМ
#### Поставить текущий выбранный блок - ПКМ
#### Выбрать блок - цифрами ЛИБО нажатием ЛКМ в UI

## Установка зависимостей

#### Для Windows
```bash
pip3 install -r requirements.txt
```

#### Для Linux
```bash
sudo apt-get install python3-pygame
```

## Запуск игры
```bash
python3 main.py
```
